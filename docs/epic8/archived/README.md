# Epic 8 Archived Documentation

This directory contains historical documents that may contain outdated or conflicting information about Epic 8 implementation status.

## Purpose

These documents are preserved for historical context but should not be used for current development decisions. They may contain:
- Optimistic completion percentages that don't match evidence
- Conflicting status reports  
- Outdated implementation plans
- Superseded architectural decisions

## Current Reference

For accurate, evidence-based status information, always refer to:
- [`../EPIC8_CURRENT_STATUS.md`](../EPIC8_CURRENT_STATUS.md) - Single source of truth
- Project root evidence files (EPIC8_HANDOFF_REPORT.md, etc.)

## Archive Criteria

Documents are moved here when they:
- Contain conflicting completion percentages
- Make claims not supported by evidence
- Have been superseded by more accurate assessments
- Provide outdated technical specifications
