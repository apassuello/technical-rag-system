# Epic 1 Validation

**Usage**: `/epic1-validate [component]`
**Examples**:
- `/epic1-validate analyzer` - Test query analyzer
- `/epic1-validate adapters` - Test all adapters
- `/epic1-validate routing` - Test routing logic
- `/epic1-validate all` - Complete Epic 1 validation

## Instructions

Run validation tests for Epic 1 components and integration.

## Validation Suites

**analyzer**:
- Classification accuracy test (>85% target)
- Feature extraction validation
- Edge case handling
- Performance benchmarks

**adapters**:
- Connection validation
- Cost tracking accuracy
- Error handling tests
- Token counting verification

**routing**:
- Strategy selection tests
- Fallback chain validation
- Cost optimization verification
- Performance overhead (<50ms)

**integration**:
- End-to-end flow test
- Configuration switching
- Epic 2 compatibility
- Cost reduction validation (>40%)
